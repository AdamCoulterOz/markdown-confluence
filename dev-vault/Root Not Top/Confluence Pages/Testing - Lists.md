
### Lists

#### Unnumbered lists

```md
- Item 1
- Item 2
  - Item 2a
  - Item 2b
```

- Item 1
- Item 2
  - Item 2a
  - Item 2b

#### Numbered Lists

```md
1. Item 1
2. Item 2
3. Item 3
   1. Item 3a
   2. Item 3b
```

1. Item 1
2. Item 2
3. Item 3
   1. Item 3a
   2. Item 3b

---

Create a _loose list_ by adding a blank line between any two list items.

```md
- Item 1

- Item 2
- Item 3
```

Will look like this:

- Item 1

- Item 2
- Item 3
